Add Facebook Like and Send buttons after each post. Automatically include Open Graph Meta Tags.


== Description == 

Add Facebook Like and Send buttons in your WordPress posts or pages. If the user is already logged into Facebook, the plugin display profile photos below the button of a user's friends who have liked the URL specified.

In addition, this plugin adds Facebook Open Graph Meta Data in your WordPress site to show the right thumbnail image, or the right title and description.

By adding Open Graph Meta tags together with the Facebook Like and Send buttons, you can get better control over how your content is displayed on Facebook.

This plugin can be the single most important factor in making your article go viral on Facebook.

Plugin by <a href="http://tutskid.com/">TutsKid | WordPress Tutorials, Themes, Plugins, and More!</a>.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Activate the plugin through the 'Plugins' menu in WordPress.

OR

Just grab the WP Facebook Like Send & Open Graph Meta plugin from the built in plugin directory and choose "Install now", hit ok, then select the "Activate Plugin" link.
